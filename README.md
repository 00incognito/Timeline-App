# Biblical Timeline App

This uses React + Mapbox GL JS to visualize Biblical events on a map timeline.

## Prerequisites

You need **Node.js** installed to run this project.

1.  **Check if Node is installed**:
    Run `node -v` in your terminal. It should print a version (e.g., `v20.x.x`).
    If not, install it from [nodejs.org](https://nodejs.org/).

2.  **Mapbox Token**:
    Make sure you've added your Mapbox public token to the `.env` file:
    ```bash
    VITE_MAPBOX_TOKEN=pk.YOUR_ACTUAL_TOKEN
    ```

## Installation

1.  Install dependencies:
    ```bash
    npm install
    ```

## Running the App

1.  Start the development server:
    ```bash
    npm run dev
    ```

2.  Open your browser to the URL shown (usually `http://localhost:5173`).
